My Finances Checker

This app allows users to gauge how healthy their monthly finances are based on the 50-30-20 budgeting method by receiving users incomes and expenses as inputs.

Stack

  • HTML
  • CSS
  • React

Live

View Site

Purpose and Goal

The idea came to me after talking to friends and family. We were talking about personal finance. When I mentioned the 50-30-20 budgeting method, which I learned after reading this article, I realized that many people haven't heard about it.

The main goal of the project was to teach users about this method and present them with an easy tool to gauge how well their monthly budget is distributed.

Web Stack and Explanation

React is my go to JavaScript library. It allows you to work with components that combines logic and UI.

So what better tool to create this project than React. I also used React Router to make it a Single Page Application (SPA).
